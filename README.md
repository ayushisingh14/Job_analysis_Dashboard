# 📊 AI & ML Job Analysis Dashboard (Power BI)

## 📌 Project Overview
This project is a Power BI dashboard designed to analyze the AI & ML job market. It provides insights into job trends, salary distribution, experience levels, and remote work opportunities.

---

## 🛠️ Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling (Star Schema)
- Excel / CSV datasets

---

## 📂 Dataset Description
The dataset consists of multiple tables:
- Dim_Companies
- Dim_Date
- Dim_Locations
- Dim_Skills
- Fact_Jobs
- Fact_Job_Skills

---

## 📈 Key Features & Insights
- 📌 Total Jobs and Monthly Job Trends
- 💰 Average Salary and Median Salary Analysis
- 📊 Salary Distribution by Experience Level
- 🌍 Location-based Job Insights
- 🏢 Company and Role Analysis
- 🧑‍💻 Skill Demand Analysis
- 🏠 Remote Job Percentage

---

## 📊 Dashboard Pages
1. Overview  
2. Location Insights  
3. Company & Role Insights  
4. Skill & Hiring Insights  

---

## 📊 Key KPIs
- Total Jobs: 10  
- Average Salary: 93.40K  
- Median Salary: 102.50K  
- Remote Jobs: 10%  

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Explore interactive dashboards using filters  

---

## 💡 Conclusion
This dashboard helps understand trends in the AI & ML job market, including salary patterns, hiring demand, and skill requirements, which can help job seekers and analysts make data-driven decisions.
